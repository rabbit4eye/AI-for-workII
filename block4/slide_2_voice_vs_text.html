Right Interface, Right Context

เลือกให้ถูก ใช้ให้เป็น: Voice vs Text

Voice ดีกว่าตอน...
ขับรถ / เดินทาง (Commute)
มือไม่ว่าง (ทำอาหาร/ออกกำลัง)
Brainstorm / คิดไม่ออก
ฝึกซ้อม Presentation
Reflection / ระบายความรู้สึก
Text ดีกว่าตอน...
ต้องการตัวเลขแม่นยำ
ต้องแก้ไข / คัดลอกไปใช้ต่อ
อยู่ในที่สาธารณะ / ประชุม
ทำงานกับเอกสารยาวๆ
เขียน Code / สูตร Excel

Voice ไม่ได้มาแทน Text แต่เป็น Interface เสริม ที่ปลดล็อก Context ใหม่ๆ
