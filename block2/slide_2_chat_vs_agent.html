Chat vs Agent

ความต่างที่เปลี่ยนวิธีทำงาน

AI Chat
  • ถาม → ตอบ
    รอคุณป้อนคำถามทีละข้อ
  • คุณต้องพิมพ์ Prompt เองทุกขั้นตอน
  • ทำงานอยู่ในกล่อง Chat เท่านั้น
  • ตอบจาก "ความจำ" ที่เรียนมา
เหมือน: พนักงานทั่วไป "สงกรานต์ปีนี้วันไหนครับ?"
VS
AI Agent
  • สั่ง → ไปทำเอง
    ลงมือทำให้จนจบงาน
  • บอกเป้าหมาย (Outcome) แค่ครั้งเดียว
  • เปิด Browser หาข้อมูลจริง Real-time
  • วิเคราะห์ เปรียบเทียบ สรุปผล
เหมือน: เลขาส่วนตัว "ช่วยจองตั๋วไปเชียงใหม่ให้หน่อย"
